## Photo de Profil LinkedIn - Fabrice Fays

### Instructions pour ajouter votre photo de profil

1. **Accédez à votre profil LinkedIn** :
   - Allez sur https://www.linkedin.com/in/fabrice-fays/
   - Cliquez sur votre photo de profil (coin supérieur droit)
   - Sélectionnez "Afficher le profil" ou "Open profile"

2. **Téléchargez votre photo** :
   - Téléchargez votre photo de profil LinkedIn (format haute qualité)
   - Le lien direct à votre photo est généralement : 
     ```
     https://media.licdn.com/dms/image/[YOUR_IMAGE_ID]/profile-displayphoto-shrink_400_400/[MORE_PARAMS]
     ```

3. **Préparez le fichier** :
   - Renommez la photo en : `fabrice-fays.jpg`
   - Optimisez le fichier aux spécifications :
     - **Format** : JPG ou PNG
     - **Ratio** : Portrait (3:4 idéalement)
     - **Dimension minimum** : 800 x 1200 pixels
     - **Poids** : Maximum 500 KB
   - Conseil : Utilisez un outil comme TinyJPG pour compresser sans perte de qualité

4. **Placez le fichier** :
   - Chemin : `public/images/fabrice-fays.jpg`
   - Fichier cible : `c:\Users\fabri\OneDrive\Bureau\POWALYZE FINAL\public\images\fabrice-fays.jpg`

5. **Vérifiez l'affichage** :
   - Attendez le rechargement automatique du navigateur
   - La photo apparaîtra sur la page https://localhost:3001/about
   - Si elle ne s'affiche pas, videz le cache navigateur (Ctrl+Shift+Suppr)

### Fallback (Image Placeholder)

Si le fichier n'existe pas ou ne se charge pas, une photo professionnelle placeholder s'affichera automatiquement depuis Unsplash.

### Support
- Formats acceptés : JPG, JPEG, PNG
- La page se met à jour automatiquement après placement du fichier
- Pas besoin de rebuild le site, juste un rafraîchissement navigateur
